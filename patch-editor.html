💡 patcher-v2.js Setup

          

            1. Drop patcher-v2.js into your game folder

            2. In index.html replace the old script tag:

              <script src="patcher-v2.js" defer>

            3. Single patch → use patch.json (same as before)

            4. Multi-patch → use patches.json (array)

            5. Add ?vf-debug to URL for the dev panel
